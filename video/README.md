# Папка для видео

Положите сюда свой видеофайл с именем **`video.mp4`**.

Итоговый путь должен быть таким:

```
video/video.mp4
```

## Как заменить видео

1. Скопируйте свой файл в эту папку и назовите его `video.mp4`
   (перезаписав/удалив эту инструкцию при желании).
2. Откройте `index.html` в браузере — видео появится в плеере после ввода пароля.

## Другое имя или формат файла

Если хотите использовать файл с другим именем или путём — измените переменную
`videoSrc` в файле `script.js` (блок `CONFIG` в самом начале):

```js
const CONFIG = {
  videoSrc: "video/my-lesson.mp4", // ← путь к вашему файлу
  videoType: "video/mp4",          // ← тип: video/mp4, video/webm и т.д.
  ...
};
```

> Формат `.mp4` (кодек H.264) рекомендуется — он воспроизводится во всех браузерах.
